# FTF_Tool
云顶之弈工具，selenium爬虫，python

最近爱上了云顶之弈，懒得每次都去查阵容。之前用过TFT Overlay工具，但是每换一次大版本就要去找资源贼麻烦。就想着自己写个爬虫，然后自己组织数据，写个软件出来。
上LOL官网发现，官网的数据很全，所以目标锁定为爬取官网的数据了。爬虫模块我用Selenium。
